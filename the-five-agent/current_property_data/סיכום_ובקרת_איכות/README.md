# סיכום_ובקרת_איכות — Agent 6 & QC

תיקייה ייעודית לפלטי הסוכנים הסופיים ולקבצי עזר לבקרת איכות.

| קובץ | תוכן |
|------|------|
| `draft_report.md` | טיוטת השומה המאורגנת (נוצרת ע"י ה-CEO בשלב AssembleReport) |
| `audit_log.md` | יומן ביקורת (SPAWN→VERIFY→GATE לכל סוכן) |
| `qc_checklist.md` | תוצאות בקרת האיכות הסופית |

**Agent 6** הוא synthesis-only — אינו קורא קבצים. קיבל את כל נתוניו מה-CEO.
**QC Agent** קורא מ-`draft_report.md` ומ-`audit_log.md` לצורך בדיקת עקביות רוחבית.
