# Agent 1: Setup & ID — PRD

## Overview

PRD and agent.md definition for Agent 1: Setup & ID, the first chapter agent in the Standard 19 appraisal system. This agent runs immediately after the Research Agent and produces chapters 1.0, 2.0, and 3.0 — the identity and general framework sections of the appraisal.

**Files:**
- `the-five-agent/agent1-setup-id-prd.md` — full PRD
- `the-five-agent/.claude/agents/agent1-setup-id.md` — agent definition for Claude Code

**Chapter scope:**
- 1.0 פרטי זיהוי הנכס (12-field property ID table)
- 2.0 פרק א': כללי (6 sub-sections: מזמין, זכויות, מועד, ביקור, הצהרה, מסמכים)
- 3.0 פרק ב': זיהוי הנכס (5 sub-sections: רישומי, פיזי, מוניציפלי, מהות, מפות)

**Gold Standard reference:** Structure derived from completed Standard 19.0 appraisal for פרויקט "חצר הנביאים", רחוב מונבז 3, דירה 101, ירושלים — גוש 30052, חלקה 176, תת-חלקה 109.
